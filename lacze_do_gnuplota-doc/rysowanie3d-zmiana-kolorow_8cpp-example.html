rysowanie3d-zmiana-kolorow.cpp

Zawiera przykład uruchomienia programu gnuplot w trybie w trybie rysowania wykresów przestrzennych 3D. Dodatkowo pokazane jest jak można zmieniać grubość rysowanej linni oraz jej kolor.

Zrzut ekranu pokazujący wynik działania programu.

Zawartość plików opisujących poszczególne powierzchnie jest taka sama jak we wcześniejszym przykładzie.

Kod programu:

#include <iostream>
#include <iomanip>
#include "lacze_do_gnuplota.hh"
using namespace std;
int main()
{
PzG::LaczeDoGNUPlota Lacze;
Lacze.ZmienTrybRys(PzG::TR_3D);
Lacze.Inicjalizuj(); // Tutaj startuje gnuplot.
Lacze.UstawZakresX(-40, 100);
Lacze.UstawZakresY(-90, 90);
Lacze.UstawZakresZ(-20, 90);
Lacze.UstawRotacjeXZ(40,60); // Tutaj ustawiany jest widok
PzG::InfoPlikuDoRysowania *wInfoPliku1 = &Lacze.DodajNazwePliku("dat/prostopadloscian1.pow");
wInfoPliku1->ZmienKolor(5);
wInfoPliku1->ZmienSzerokosc(4);
Lacze.Rysuj(); // Teraz powinno pojawic sie okienko gnuplota
// z rysunkiem, o ile istnieje plik "prostopadloscian1.pow"
cout << "Nacisnij ENTER, aby zobaczyc prostopadloscian nr 2 ... " << flush;
cin.ignore(10000,'\n');
PzG::InfoPlikuDoRysowania *wInfoPliku2 = &Lacze.DodajNazwePliku("dat/prostopadloscian2.pow");
wInfoPliku1->ZmienKolor(2);
wInfoPliku1->ZmienSzerokosc(1);
wInfoPliku2->ZmienKolor(5);
wInfoPliku2->ZmienSzerokosc(4);
Lacze.Rysuj(); // Teraz powinno pojawic sie okienko gnuplota
// z rysunkiem, o ile istnieje plik "prostopadloscian2.pow"
cout << "Nacisnij ENTER, aby zobaczyc prostopadloscian nr 3 ... " << flush;
cin.ignore(10000,'\n');
PzG::InfoPlikuDoRysowania *wInfoPliku3 = &Lacze.DodajNazwePliku("dat/prostopadloscian3.pow");
wInfoPliku2->ZmienKolor(3);
wInfoPliku2->ZmienSzerokosc(2);
wInfoPliku3->ZmienKolor(5);
wInfoPliku3->ZmienSzerokosc(4);
Lacze.Rysuj(); // Teraz powinno pojawic sie okienko gnuplota
// z rysunkiem, o ile istnieje plik "prostopadloscian3.pow"
cout << "Nacisnij ENTER, aby zakonczyc ... " << flush;
cin.ignore(10000,'\n');
}
