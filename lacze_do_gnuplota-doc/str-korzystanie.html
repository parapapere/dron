Rysowanie brył - wykorzystywanie plików

Moduł pozwala na utworzenie zbioru plików zawierających opis dowolnych brył. Format opisu bryły ma postać zbioru współrzędnych kolejnych wierzchołków. Kolejność jak też sposób ich podawania musi być zgodny z wymogami narzucownymi przez program gnuplot dla przypadku wizualizacji powierzchni. Przedstawiony poniżej plik "korpus.dat" zawiera opis prostopadłościanu.

   0 -20   0 
  30 -20  10
  30  20  10
   0  20   0
 #

   0 -20   0 
  30 -20 -10
  30  20 -10
   0  20   0
 #

   0 -20   0 
 -30 -20 -10
 -30  20 -10
   0  20   0
 #

   0 -20   0 
 -30 -20  10
 -30  20  10
   0  20   0
 #

   0 -20   0 
  30 -20  10
  30  20  10
   0  20   0
 #

Ilość znaków "białych" między poszczególnymi liczbami nie jest istotna. Przykładowy rysunek utworzony za pomocą programu gnuplot przedstawiony jest poniżej.

Przykład rysunku prostopadłościanu.

Najprostszy program, który umożliwia rysowanie na podstawie danych zawartych w przedstawionym powyżej pliku może mieć postać.

#include <iostream>
#include <iomanip>
#include "lacze_do_gnuplota.hh"
using namespace std;
int main()
{
PzG::LaczeDoGNUPlota Lacze;
char ZnakKonca;
Lacze.ZmienTrybRys(PzG::TR_3D);
Lacze.DodajNazwePliku("korpus.dat");
Lacze.Inicjalizuj(); // Tutaj startuje gnuplot.
Lacze.Rysuj(); // Teraz powinno pojawic sie okienko gnuplota
// z rysunkiem, o ile istnieje plik "korpus.dat"
cout << "Nacisnij ENTER, aby zakonczyc." << endl;
cin >> noskipws >> ZnakKonca;
}
Uwaga
Nazwy plików, z których będzie brany opis rysowanej powierzchni zadajemy tylko raz. Błędem jest zadawanie tych samych nazw przed każdorazowym rysowaniem. Nazwy plików pamiętane są przez cały czas. Można jednak je usunąć korzystając z metody PzG::LaczeDoGNUPlota::UsunWszystkieNazwyPlikow.

Aby powiększyć obiekt na rysunku, możemy w tym celu użyć zmiany zakresu podziałki. Służą do tego metody PzG::LaczeDoGNUPlota::UstawZakresX, PzG::LaczeDoGNUPlota::UstawZakresY oraz PzG::LaczeDoGNUPlota::UstawZakresZ. Chcąc uniknąć zniekształceń zmian należy dokonać równomiernie wzdłuż wszystkich osi. Obrazuje to poniże przedstawiony kod programu.

#include <iostream>
#include <iomanip>
#include "lacze_do_gnuplota.hh"
using namespace std;
#define POWIEKSZ 6
int main()
{
PzG::LaczeDoGNUPlota Lacze;
char ZnakKonca;
Lacze.ZmienTrybRys(PzG::TR_3D);
Lacze.DodajNazwePliku("korpus.dat");
Lacze.Inicjalizuj(); // Tutaj startuje gnuplot.
Lacze.UstawZakresX(Lacze.Xmin()/POWIEKSZ,Lacze.Xmax()/POWIEKSZ);
Lacze.UstawZakresY(Lacze.Ymin()/POWIEKSZ,Lacze.Ymax()/POWIEKSZ);
Lacze.UstawZakresZ(Lacze.Zmin()/POWIEKSZ,Lacze.Zmax()/POWIEKSZ);
Lacze.Rysuj(); // Teraz powinno pojawic sie okienko gnuplota
// z rysunkiem, o ile istnieje plik "korpus.dat"
cout << "Nacisnij ENTER, aby zakonczyc." << endl;
cin >> noskipws >> ZnakKonca;
}

Zmianę punktu patrzenia na scenę można osiągnąć wykorzystując metodę PzG::LaczeDoGNUPlota::UstawRotacjeXZ.

int main()
{
PzG::LaczeDoGNUPlota Lacze;
char ZnakKonca;
Lacze.ZmienTrybRys(PzG::TR_3D);
Lacze.DodajNazwePliku("korpus.dat");
Lacze.Inicjalizuj(); // Tutaj startuje gnuplot.
Lacze.UstawZakresX(Lacze.Xmin()/POWIEKSZ,Lacze.Xmax()/POWIEKSZ);
Lacze.UstawZakresY(Lacze.Ymin()/POWIEKSZ,Lacze.Ymax()/POWIEKSZ);
Lacze.UstawZakresZ(Lacze.Zmin()/POWIEKSZ,Lacze.Zmax()/POWIEKSZ);
Lacze.UstawRotacjeXZ(40,60); // Tutaj zadajemy zmianę punktu obserwacji.
Lacze.Rysuj(); // Teraz powinno pojawic sie okienko gnuplota
// z rysunkiem, o ile istnieje plik "korpus.dat"
cout << "Nacisnij ENTER, aby zakonczyc." << endl;
cin >> noskipws >> ZnakKonca;
}

Istnieje możliwość operowania wieloma widokami jednocześnie. Przykład programowej realizacji takiej opcji demonstruje poniższy kod.

int main()
{
PzG::LaczeDoGNUPlota Lacze, DrugieLacze;
char ZnakKonca;
Lacze.ZmienTrybRys(PzG::TR_3D);
Lacze.DodajNazwePliku("korpus.dat"); // Lista plików jest wspólna dla
// wszystkich łączy. Nazwe wystarczy
// dodać tylko poprzez jedno z łączy.
Lacze.Inicjalizuj(); // Tutaj startuje gnuplot.
DrugieLacze.Inicjalizuj();
Lacze.UstawZakresX(Lacze.Xmin()/POWIEKSZ,Lacze.Xmax()/POWIEKSZ);
Lacze.UstawZakresY(Lacze.Ymin()/POWIEKSZ,Lacze.Ymax()/POWIEKSZ);
Lacze.UstawZakresZ(Lacze.Zmin()/POWIEKSZ,Lacze.Zmax()/POWIEKSZ);
Lacze.UstawRotacjeXZ(40,60);
DrugieLacze.UstawRotacjeXZ(80,10);
Lacze.Rysuj(); // Teraz powinno pojawic sie okienko gnuplota
DrugieLacze.Rysuj(); // z rysunkiem, o ile istnieje plik "korpus.dat"
cout << "Nacisnij ENTER, aby zakonczyc." << endl;
cin >> noskipws >> ZnakKonca;
}
